# Agent Rules

## Code Review Required

Every commit on `main` or any `feature/*` / `fix/*` branch that modifies
source code (`lib/`, `test/`, `example/`), documentation (`README`,
`CHANGELOG`), or configuration (`Rakefile`, `*.gemspec`) **must** be
reviewed by a sub-agent, the user, or another reviewer before merging.

Transient/working files such as `REVIEW.md` and other scratch notes in
the repository root are exempt from this rule.
